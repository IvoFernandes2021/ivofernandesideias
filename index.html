Comando: LINE

O comando LINE é utilizado para criar segmentos de reta.

Como usar:

  1. Digite LINE ou clique no ícone correspondente na barra de ferramentas.
  2. Especifique o primeiro ponto clicando na área de desenho ou inserindo as coordenadas.
  3. Especifique o próximo ponto. Continue especificando pontos para criar segmentos de reta consecutivos.
  4. Pressione Enter para finalizar o comando.

Comando: CIRCLE

O comando CIRCLE é usado para criar círculos.

Como usar:

  1. Digite CIRCLE ou clique no ícone correspondente.
  2. Especifique o centro do círculo clicando ou inserindo coordenadas.
  3. Especifique o raio do círculo clicando ou inserindo o valor.
